## Deployment
<code>gradle clean tomcatRunWar</code>

[link](https://github.com/bmuschko/gradle-tomcat-plugin)

## API Documentation
API Doc is generated by swagger, visit http://localhost:8080/documentation/swagger-ui.html

## Quality

### Following tools are integrated for code quality analysis
* checkstyle
* jacoco
* sonar

Code are formatted in google style by using schema under checkstyle/style folder.

Run following gradle command to start analysis:

<code>gradle clean build jacocoTestReport sonarqube</code>

By default, checkstyle's severity is error, build will fail if checkstyle fails.

### Quick setup SonarQube in docker container
<code>$ docker run -d --name sonarqube -p 9000:9000 -p 9092:9092 sonarqube:6.7.3</code>

By default you can login as admin with password admin, see [details](https://hub.docker.com/_/sonarqube/).
